## Este proyecto implementa un sistema de gestión de biblioteca utilizando Java y los principios de Programación Orientada a Objetos (POO), incluyendo herencia y polimorfismo.

## Funcionalidades
- Registro de libros
- Registro de usuarios
- Préstamo de libros
- Devolución de libros
- Búsqueda de libros por título o autor
- Visualización de libros disponibles
- Visualización de usuarios registrados
- Control de máximo 3 préstamos por usuario
- Validación de disponibilidad de libros


## 🧠 Conceptos aplicados
- Encapsulamiento
- Herencia (Persona → Usuario)
- Polimorfismo (sobrescritura y sobrecarga)
- Clases y objetos
